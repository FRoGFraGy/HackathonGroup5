# BKK Odor Early Warning System 🌬️🚨
> **ระบบคาดการณ์กลิ่นเคมีและวางแผนอพยพภัยพิบัติอัจฉริยะแบบเรียลไทม์ (มุมมองผู้ว่าฯ กทม.)**

ระบบแดชบอร์ดอัจฉริยะสำหรับผู้บริหารและผู้ว่าราชการกรุงเทพมหานคร เพื่อใช้ในการติดตาม จำลองสถานการณ์การแพร่กระจายของกลิ่นเคมี/แก๊สพิษ และสั่งการอพยพประชาชนอย่างมีประสิทธิภาพและทันท่วงทีเมื่อเกิดภัยพิบัติ

---

## 🌟 ฟีเจอร์เด่น (Key Features)

### 1. ระบบจำลองสถานการณ์ฉุกเฉิน (Incident Simulator)
- จำลองการรั่วไหลจากแหล่งกำเนิด 3 ประเภทหลัก: **โรงงานเคมี**, **โรงบำบัดน้ำเสีย**, และ **บ่อขยะ**
- เลือกสถานการณ์ด่วน (Preset Scenarios) เช่น:
  - แก๊สเคมีรั่วไหล ลาดกระบัง-กิ่งแก้ว (สาร Styrene Monomer)
  - กลิ่นเหม็นไหม้ บ่อขยะหนองแขม (ก๊าซ H₂S)
  - สารเคมีโกดังสินค้า ท่าเรือคลองเตย (แอมโมเนีย)

### 2. โมเดลจำลองทิศทางลมแบบอินเตอร์แอคทีฟ (Meteorological Simulation)
- ปรับเปลี่ยนมุมทิศทางลม (Wind Direction) และความเร็วลม (Wind Speed) ได้แบบเรียลไทม์
- เข็มทิศนำทางแสดงทิศทางลมพัดไปอย่างชัดเจน

### 3. แผนที่การฟุ้งกระจายของกลุ่มแก๊ส (Odor & Gas Risk Plume Map)
- แสดงกลุ่มแก๊สและการแพร่กระจายตัวแบบกราฟิกสวยงามด้วย **HTML5 Canvas Animation**
- สามารถลากและวางเป้าเล็งจุดเกิดเหตุ (<i class="fa-solid fa-crosshairs"></i>) เพื่อเปลี่ยนพิกัดการจำลองจุดรั่วไหลได้ทันทีบนแผนที่
- คำนวณช่วงเวลาคาดการณ์ล่วงหน้า 30 นาที (30-Min Forecast Slider)

### 4. ระบบวิเคราะห์ช่วยเหลือการตัดสินใจ (Decision Support System)
- **Warning Zones:** วิเคราะห์และระบุเขตกรุงเทพฯ ที่จะได้รับผลกระทบจากกลุ่มแก๊สพิษตามแนวทิศทางลม
- **Evacuation Routes:** แนะนำเส้นทางหลักที่ปลอดภัยและแจ้งเตือนจุดที่ควรหลีกเลี่ยง
- **Recommended Shelters:** แนะนำจุดพักพิงที่ปลอดภัย (Safe Zones) ที่อยู่นอกแนวทิศทางก๊าซพิษ

### 5. ศูนย์สั่งการผู้ว่าราชการกรุงเทพฯ (Direct Command Console)
- ปุ่มสั่งการด่วน: **แจ้งเตือน SMS/LINE** ไปยังประชากรกลุ่มเสี่ยง, **สั่งเริ่มแผนอพยพ**, และ **ส่งหน่วยพ่นละอองน้ำ** เพื่อสลายสารเคมีในอากาศ

## 🛠️ เทคโนโลยีที่เลือกใช้ (Technologies Used)

- **Frontend Structure:** HTML5 (Semantic Elements)
- **Styling:** Tailwind CSS (ผ่าน CDN สำหรับความรวดเร็วในการพัฒนา)
- **Icons:** FontAwesome v6
- **Typography:** Google Fonts (Mitr & Sarabun)
- **Graphics & Animation Engine:** Vanilla JavaScript ร่วมกับ HTML5 Canvas สำหรับรัน Particle และกรวยกระจายแก๊สพิษแบบเรียลไทม์

## 💻 วิธีการรันโปรเจค (How to Run)

เนื่องจากระบบนี้พัฒนาขึ้นด้วยโครงสร้างแบบ Static Web Application ที่ไร้ dependencies ภายนอกที่ซับซ้อน คุณสามารถรันโปรเจคนี้ได้ง่ายๆ 2 วิธี:

### วิธีที่ 1: ดับเบิ้ลคลิกเปิดไฟล์โดยตรง
1. ไปยังโฟลเดอร์ของโปรเจค
2. ดับเบิ้ลคลิกที่ไฟล์ `smart_evacuation_dashboard.html` เพื่อเปิดใช้งานบนเว็บเบราว์เซอร์ได้ทันที

### วิธีที่ 2: รันผ่าน Local Web Server (แนะนำ)
หากต้องการรันผ่าน Web Server เพื่อความสมบูรณ์แบบในการโหลดสคริปต์และลดปัญหาระบบ Security ของบราวเซอร์:

**ใช้งานผ่าน Python:**
```bash
python -m http.server 8000
```
จากนั้นเปิดเบราว์เซอร์แล้วเข้าไปที่: [http://localhost:8000/smart_evacuation_dashboard.html](http://localhost:8000/smart_evacuation_dashboard.html)

**ใช้งานผ่าน Node.js (npx):**
```bash
npx http-server -p 8000
```
จากนั้นเปิดเบราว์เซอร์แล้วเข้าไปที่: [http://localhost:8000/smart_evacuation_dashboard.html](http://localhost:8000/smart_evacuation_dashboard.html)
